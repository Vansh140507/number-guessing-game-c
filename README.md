# 🎮 Number Guessing Game (C)
 
A simple command-line number guessing game built in C. The computer picks a random number between 1 and 100, and you have to guess it!
 
## 📋 How It Works
 
- The program generates a **random number between 1 and 100**
- You enter your guess
- The program tells you if your guess is **Higher** or **Lower**
- Keep guessing until you find the correct number!
